## 一、技术指标数学原理

### 1.1 趋势类指标

**SMA（简单移动平均）**

\[
\text{SMA}_N(t) = \frac{1}{N} \sum_{i=0}^{N-1} P_{t-i}
\]

最基础的平滑工具。短周期灵敏但噪音多，长周期稳健但滞后大。交叉系统利用短/长周期的速度差。

**EMA（指数移动平均）**

\[
\text{EMA}_t = \alpha P_t + (1-\alpha) \text{EMA}_{t-1}, \quad \alpha = \frac{2}{N+1}
\]

相比 SMA，EMA 对新数据赋予指数衰减的更高权重，反应更快。MACD 的核心组件。

**MACD（指数平滑异同移动平均线）**

\[
\begin{aligned}
\text{DIF}_t &= \text{EMA}(P, 12) - \text{EMA}(P, 26) \\[2pt]
\text{DEA}_t &= \text{EMA}(\text{DIF}, 9) \\[2pt]
\text{MACD柱}_t &= \text{DIF}_t - \text{DEA}_t
\end{aligned}
\]

信号逻辑：DIF 上穿 DEA（金叉）买入，下穿（死叉）卖出。柱状图反映趋势加速度。

### 1.2 动量类指标

**RSI（相对强弱指标）— Wilder (1978)**

\[
\text{RSI} = 100 - \frac{100}{1 + RS}, \quad RS = \frac{\text{Avg Gain}_N}{\text{Avg Loss}_N}
\]

其中 Avg Gain/Loss 使用 Wilder 平滑（类似 EMA，\(\alpha=1/N\)）。值域 [0,100]，>70 超买，<30 超卖。A股实际最优阈值往往更极端（如 20/80）。

**Stochastic（随机指标）— Lane (1950s)**

\[
\%K = \frac{C_t - \text{Low}_N}{\text{High}_N - \text{Low}_N} \times 100
\]

衡量收盘价在 N 日高低区间中的相对位置。>80 高位，<20 低位。

**CCI（商品通道指数）— Lambert (1980)**

\[
\text{CCI} = \frac{TP - \text{SMA}(TP)}{0.015 \cdot \text{MeanDeviation}}, \quad TP = \frac{H+L+C}{3}
\]

常数 0.015 使 ~75% 的值落在 [-100, +100] 之间。突破 ±100 为极端信号。

### 1.3 波动率类指标

**Bollinger Bands — Bollinger (1980s)**

\[
\begin{aligned}
\text{中轨} &= \text{SMA}(P, 20) \\
\text{上轨} &= \text{中轨} + 2\sigma_{20} \\
\text{下轨} &= \text{中轨} - 2\sigma_{20}
\end{aligned}
\]

带宽反映波动率。收窄（squeeze）预示突破；触及上下轨暗示回归或趋势延续。

**ATR（平均真实波幅）— Wilder (1978)**

\[
TR = \max(H_t - L_t,\ |H_t - C_{t-1}|,\ |L_t - C_{t-1}|)
\]

\[
\text{ATR} = \text{WilderSmooth}(TR, 14)
\]

不含方向信息，仅衡量波动幅度。广泛用于动态止损和仓位计算。

### 1.4 成交量类指标

**OBV（能量潮）— Granville (1963)**

\[
\text{OBV}_t = \text{OBV}_{t-1} + \begin{cases}
+V_t & \text{if } C_t > C_{t-1} \\
0 & \text{if } C_t = C_{t-1} \\
-V_t & \text{if } C_t < C_{t-1}
\end{cases}
\]

量在价先：OBV 走势领先于价格变动，背离是重要信号。

## 二、经典量化策略体系

### 2.1 趋势跟踪（Trend Following）

**逻辑**：顺应市场趋势，斩断亏损、让利润奔跑。正偏态分布——多次小亏换取少数大赢。

**实现**：
- 双均线交叉（快线 > 慢线 → 做多）
- 唐奇安通道突破（价格突破 N 日最高点 → 入场）
- 海龟交易法则（20日/55日突破 + ATR 仓位管理）

**适用**：强趋势市。在震荡市中反复止损是致命伤。

### 2.2 均值回归（Mean Reversion）

**逻辑**：价格偏离均值后会回归。在极端值反向建仓。

**实现**：
- RSI 超卖买入 / 超买卖出
- Bollinger 下轨买入 / 上轨卖出
- Z-Score = \((P - \text{SMA})/\sigma\)，|Z| > 2 入场

**适用**：震荡市。但在强趋势中抄底会持续亏损。

### 2.3 配对交易（Pairs Trading）

**逻辑**：两只高度相关的股票，当价差偏离历史均值时做多弱势、做空强势，赌价差回归。**市场中性**，不依赖大盘方向。

**数学基础**：协整检验（Engle-Granger, Johansen），验证价差的均值回归性。

### 2.4 统计套利（Statistical Arbitrage）

配对交易的推广：对一篮子资产按某个因子打分排名，做多得分最高的、做空得分最低的。核心步骤：

1. **估计预期收益**：用 IC（信息系数）衡量因子预测力，IC 衰减慢的因子最有价值
2. **最小化方差**：约束优化或行业中性化
3. **最小化交易成本**：设置无交易缓冲区

### 2.5 动量策略（Momentum）

**逻辑**：过去涨的继续涨，过去跌的继续跌。分横截面动量（买排名高的股票）和时间序列动量（买自己历史上表现好的）。

**关键发现**（Jegadeesh & Titman, 1993）：3-12 个月动量的超额收益最显著，但会有动量崩溃风险（momentum crash）。

### 2.6 策略对比总结

| 策略 | 持仓方向 | 市场依赖 | 胜率特征 | 最佳市场 |
|---|---|---|---|---|
| 趋势跟踪 | 单向 | 依赖趋势 | 低胜率、高盈亏比 | 强趋势市 |
| 均值回归 | 单向 | 波动 | 高胜率、低盈亏比 | 震荡市 |
| 配对交易 | 多空对冲 | 中性 | ~50% | 任意（需协整对） |
| 统计套利 | 多空篮子 | 近中性 | 略高于50% | 任意 |
| 横截面动量 | 多空 | 可对冲 | 略高于50% | 趋势延续期 |

## 三、多因子体系与 Alpha 因子

### 3.1 WorldQuant Alpha101

Zura Kakushadze (2015) 发布 101 个公式化 Alpha 因子，只用 OHLCV + VWAP 数据，平均持仓 0.6-6.4 天。核心算子：

| 算子 | 含义 | 示例 |
|---|---|---|
| `rank(x)` | 横截面排名归一化 | 去量纲 |
| `ts_rank(x,d)` | 时序排名 | 趋势强度 |
| `delta(x,d)` | d 日差分 | 变化率 |
| `correlation(x,y,d)` | 滚动相关系数 | 背离检测 |
| `decay_linear(x,d)` | 线性衰减加权平均 | 近期优先 |

**按投资逻辑分类**：
- **均值回归型**（~25个）：Alpha#004, #008, #009, #024, #029
- **量价背离型**（~20个）：Alpha#002, #003, #006, #012, #013
- **蜡烛图形态型**（~15个）：Alpha#018, #028, #041, #060, #101
- **波动率条件型**（~10个）：Alpha#034, #040, #052
- **动量型**（~10个）：Alpha#001, #005, #033, #048

### 3.2 国泰君安 Alpha191

2017 年国泰君安发布 191 个 A 股短周期 Alpha 因子，其中 ~70 个改编自 Alpha101。专门针对 A 股市场特性（T+1、涨跌停、散户主导）优化。

### 3.3 A 股关键因子（学术文献汇总）

基于多篇 2024-2025 年论文的因子重要性排序：

1. **价格动量**：短期反转（1-5 天）+ 中期动量（3-12 月）
2. **换手率**：A 股最显著的特质波动率代理变量
3. **波动率**：低波动异象在 A 股同样存在
4. **市值**：小市值效应（但近年衰减）
5. **估值**：EP（盈利收益率）、BP（账面市值比）
6. **流动性**：Amihud 非流动性指标
7. **技术形态**：Alpha101 中的蜡烛图因子

### 3.4 Alpha101 之后：因子研究的十年演进（2015-2025）

**为什么 Alpha101 仍然重要但不够用？**

Alpha101 的核心贡献是证明了「只用 OHLCV + VWAP 就能写出有预测力的公式化因子」。但 2015 年至今发生了三件事：

1. **因子拥挤**：公开的 Alpha101 因子在美股已严重衰减。WorldQuant 本身早已不依赖这 101 个 — 它的 Brain 平台已收到全球 10 万+ 参与者的数百万个 alpha 提交，但这些是闭源的竞赛产物
2. **因子范式迁移**：手工编写公式 → ML 自动挖掘 → LLM 辅助生成 → 端到端深度学习
3. **A 股特殊性凸显**：美股的因子到了 A 股需要大量适配（T+1、涨跌停、散户结构、政策冲击），催生了大量 A 股原生因子研究

**WorldQuant 之后学术界公开的因子集：**

| 来源 | 年份 | 因子数 | 特点 |
|---|---|---|---|
| Alpha101 (Kakushadze) | 2015 | 101 | 纯 OHLCV+VWAP，开创性 |
| Alpha191 (国泰君安) | 2017 | 191 | ~70 个改编自 Alpha101，A 股适配 |
| ML-Enhanced (arXiv 2507) | 2025 | 500-1000 | Alpha101 扩展 + 微观结构信号，PyTorch 张量计算 |

### 3.5 2024-2025 新一代因子挖掘范式

**范式一：LLM + 强化学习联合发现公式化 Alpha**

*中科院 (Springer, 2025) — LLM-Assisted Formulaic Alpha Discovery*

用 LLM 的金融领域知识「热启动」因子搜索空间，再用强化学习交替迭代跳出局部最优。生成的是**显式数学公式**（非黑箱神经网络），天然可解释。

**核心结果**：平均 IC 达 0.0515，较纯 RL 基线提升 **75%**，累计超额收益翻倍。

这可能是手工因子和 DL 因子之间的最优折中 — 既有公式化因子的透明度，又有自动搜索的广度。

**范式二：神经网络 Alpha-Beta 协同挖掘（ABCM）**

*东方证券 (2024.12) — ABCM: 基于神经网络的 Alpha 和 Beta 因子协同挖掘*

将风险因子（Beta）和超额收益因子（Alpha）在同一个神经网络框架中联合学习，强制两者正交。解决了传统做法中「先拆 Beta 再找 Alpha」的信息损失。

**回测结果**（A 股 2017 年以来）：
- 协同 Alpha 因子年化超额收益 **36.55%**，最大回撤仅 5.86%
- 风险模型解释力：沪深 300 R² 达 **47.02%**，中证 500 R² 达 **34.28%**
- 与传统 AI 价量因子相关性极低，组合后 RankIC 升至 **16.39%**
- 中证 500 指增组合 IR 从 2.79 提升到 **3.30**，中证 1000 从 3.83 到 **4.25**

**范式三：分析师预期因子（正交 Alpha 源）**

*华泰证券 (2024.12) — 博采众长：分析师预期类因子初探*

从分析师一致预期数据中提炼三个子因子：

| 因子 | RankIC | 年化超额 |
|---|---|---|
| 异常覆盖因子（剥离市值/动量/换手后） | 2.34% | 6.59% |
| 优化评级因子（选有「异象捕捉能力」的分析师） | 2.26% | 5.17% |
| 优化盈余修正因子（去动量后的创新修正） | 3.99% | 9.55% |
| **等权合成** | **4.27%** | **10.55%** |

**关键发现**：该因子与 AI 价量因子的相关性仅 **0.03**，几乎完全正交 — 意味着叠加使用能显著提升组合的夏普比率。

*东方证券 (2023-2024) — 分析师研报类 Alpha 增强*：12 个分析师相关因子合成后，RankIC 6.45%，多空年化收益 29%，夏普 3.6，月胜率 85%。

**范式四：风险注意力因子挖掘**

*东方证券 (2024.05) — Risk-Attention 机制*

在传统 GRU/LSTM 基础上引入风险注意力 — 用风险因子构建 Query-Key 矩阵，让模型学习股票间的自适应关联结构。

**周度调仓回测**（2018-2024）：RankIC **0.106**，年化超额 **40.3%**，夏普 **5.23** — 碾压传统 GRU/MLP。

**范式五：生成式因子模型（VAE / KAN-Autoencoder）**

2024 年两篇重要工作：
- **NeuralFactors** (arXiv:2408.01499)：用 VAE 同时学习因子暴露和因子收益，在协方差估计和投资组合优化上优于传统方法，且保持与经典因子模型的数学联系
- **KAN-Autoencoder** (arXiv:2408.02694)：用 Kolmogorov-Arnold Network 替代 MLP 构建自编码器，因子暴露可解释性远优于传统神经网络

### 3.6 因子拥挤、衰减与择时

因子研究的热点正在从「发现新因子」转向「管理现有因子」：

- **因子 Beta 化**：曾经的 Alpha 因子（如低波动、动量）正在变成 Beta — 它们的收益更多来自系统性风险暴露而非真正的 alpha
- **因子相关性剧变**：Bloomberg (2025.08) 研究显示，A 股 Value-Momentum 相关性从 0.78 的历史高点跌回零附近，Value-Beta 相关性创 -0.65 历史新低 — 相关性裂变本身创造战术性配置机会
- **因子择时框架**（华安证券 2024）：结合宏观（货币信用周期 + 宏观事件）和内生（因子动量、换手率 IC、离散度）信号，沪深 300 指增组合年化超额从 12.73% 提升到 **15.32%**，最大回撤从 -7.3% 降至 -5.3%

### 3.7 对我们的启示：日线系统可落地的因子路线

考虑到我们目前的数据基础设施（日线 OHLCV，约 6 年 A 股数据），按优先级排序：

| 优先级 | 因子来源 | 所需数据 | 预期增量 |
|---|---|---|---|
| ⭐⭐⭐ | Alpha191 中 A 股适配的核心因子 | 日线 OHLCV（已有） | 中等 |
| ⭐⭐⭐ | 构建后 Alpha101 时代的新量价因子（截面/时序 ranking + 波动率条件） | 日线 OHLCV（已有） | 中等 |
| ⭐⭐ | 分析师预期因子（需额外数据源） | 分析师一致预期（需采集） | 高（正交性好） |
| ⭐⭐ | LLM+RL 辅助因子搜索 | 日线 OHLCV + 本地 GPU | 高 |
| ⭐ | ABCM Alpha-Beta 协同模型 | 日线 OHLCV + 中证/沪深指数成分股权重 | 高（复杂度高） |
| ⭐ | 基本面因子（EP/BP/ROE 等） | 财报数据（需采集） | 中等 |

## 四、机器学习在量化交易中的前沿应用

### 4.1 2024-2025 学术前沿

**LLM + RL 公式化 Alpha 发现** (Springer, 2025)：  
中国科学院的混合框架：LLM 提供金融领域知识「热启动」因子搜索，RL 交替迭代跳出局部最优。生成的 alpha 仍是显式数学公式，可解释性强。**平均 IC 达 0.0515，较纯 RL 提升 75%。**

**ABCM Alpha-Beta 协同挖掘** (东方证券, 2024)：  
神经网络联合学习风险因子（Beta）和超额收益因子（Alpha），强制二者正交，避免信息损失。A 股回测：Alpha 因子年化超额 **36.55%**，最大回撤仅 5.86%。与现有价量 AI 因子几乎零相关。

**Risk-Attention 因子挖掘** (东方证券, 2024)：  
用风险因子构建 Query-Key 注意力矩阵，捕捉股票间自适应关联。A 股周度调仓：RankIC **0.106**，年化超额 **40.3%**，夏普 **5.23**。

**MSIF-OEM 框架** (Expert Systems with Applications, 2025)：  
多源信息融合 + 在线集成学习。融合 OHLCV、限价订单簿（LOB）、逐笔成交（TAQ）数据，用并行 Transformer 编码器处理，在线集成应对因子衰减。

**DE-CS-RFA 模型** (Pacific-Basin Finance Journal, 2025)：  
110 个异质特征 + 随机森林-AdaBoost 集成 + 代价敏感学习 + 差分进化优化。A 股 2008-2024 回测。

**ML-Enhanced Multi-Factor** (arXiv, 2025)：  
500-1000 个因子（Alpha101 扩展 + 微观结构信号），PyTorch 张量化计算。**年化收益 ~20%，夏普 >2.0**。关键创新：几何布朗运动数据增强 + 横截面中性化。

**NeuralFactors VAE** (arXiv, 2024)：  
变分自编码器学习因子暴露和因子收益的联合分布，保持与经典因子模型的联系。在协方差估计和组合优化上优于传统方法。

**KAN-Autoencoder 因子模型** (Semantic Scholar, 2024)：  
Kolmogorov-Arnold Network 替代 MLP，非线性逼近因子暴露的同时保持可解释的潜在因子结构。

**Factor-GAN** (PLoS ONE, 2024)：  
70 个公司特征 + GAN 生成多因子定价。A 股多空组合年化 23.52%，夏普 1.29。

### 4.2 推荐 ML 技术路线

| 阶段 | 方法 | 用途 |
|---|---|---|
| 特征工程 | Alpha101/191 因子计算 + 自定义量价因子 | 将原始价量数据转为标准化因子 |
| 特征扩充 | 分析师一致预期数据采集 | 正交 Alpha 源，相关性 <0.05 |
| 单模型 | XGBoost / LightGBM | 因子→收益方向预测，提供特征重要性 |
| 因子搜索 | LLM + RL 辅助公式化因子发现 | 自动生成可解释的新因子公式 |
| 集成 | Stacking / Ensemble | 多模型融合，降低过拟合 |
| 深度 | Transformer / LSTM / Risk-Attention | 时序依赖建模，多尺度特征交互 |
| 增强 | GAN 数据增强 / RL 仓位管理 | 小样本学习、动态仓位 |

### 4.3 关键注意事项

- **过拟合**：A 股日线数据仅 ~6 年 ~1500 个交易日，样本量极为有限。必须做 walk-forward 样本外检验，不能单次随机切分
- **因子衰减**：A 股 Alpha 因子 IC 半衰期通常只有 3-6 个月，需要持续监控 IC 序列并轮换失效因子
- **交易成本**：T+1 + 涨跌停 + 佣金印花税，意味着日频调仓部分的实际收益要打 7-8 折
- **截面中性化**：去除市值、行业等共线性，避免暴露于 unintended risk
- **因子正交性**：新增因子前先检查与现有因子的相关性，相关 >0.7 的因子边际贡献有限

## 五、可行性优先级路线图

### 阶段一：因子库建设（当前 → 1-2周）
- 实现 Alpha101 中 30+ 个核心因子（均值回归型 + 量价背离型 + 动量型优先）
- 补充 Alpha191 中 A 股适配但 Alpha101 没有的因子
- 构建截面中性化流程（市值 / 行业去偏）
- 因子 IC 监控管线（逐日 IC、ICIR、IC 衰减曲线）

### 阶段二：单模型预测（2-4周）
- XGBoost/LightGBM 预测次日涨跌方向
- 横截面选股：每天选得分最高的 N 只
- 回测对比基线策略，计算 RankIC

### 阶段三：正交 Alpha 扩展（4-6周）
- 采集分析师一致预期数据（如有数据源）
- 实现分析师预期因子（异常覆盖 / 盈余修正）
- 验证与价量因子的正交性
- LLM + RL 辅助搜索新量价因子公式

### 阶段四：集成与优化（6-10周）
- 多模型 Stacking、SHAP 特征选择
- 仓位优化（风险平价、Kelly 公式）
- 因子择时信号引入
- 实盘模拟验证

策略版本号采用语义化版本号（准语义），从 1.00 起步。

**版本号规则：**
- 小调整（如参数微调、因子权重调整）：版本号 +0.01（如 1.00 → 1.01）
- 较大调整（如新增因子、修改算法、更换模型）：版本号 +0.1（如 1.00 → 1.10）
- 进阶：模拟盘运行效果满意 → 2.00 版本（代表可实盘操作）

**Why:** 用户希望在本地量化平台上建立清晰的版本演进轨道，方便回溯和对比不同版本的表现。

**How to apply:** 每次修改策略配置或代码后，在 strategy_versions 表中新增版本记录，版本号按上述规则递增。

## v4.0 最终结论（60+ 轮验证）

### 纯多头排序的天花板

经过 60+ 轮进化（3 批 × 20-30 轮），纯多头顶部选股在没有市场择时的情况下：
- 最佳年化：+16.7%（3 年回测，扣费后）
- 最佳 MDD：15.8%
- 最佳夏普：0.68
- 无法稳定达到 40%/15% 目标

### 核心 Alpha 结构

`mul(amount, X)` — 成交额与某个指标的乘积交互，是所有正收益因子的共同结构：
- `amount;high;mul` → -2.5%（逼近盈亏线）
- `amount;@atr_14;mul` → +16.7%（最佳）
- `amount;@rsi_14;mul` → +41.6%（6 个月回测，3 年回测不成立）

### 已验证的管线质量

- 6 年数据（3 年训练 + 3 年回测）
- 无未来函数（lu_next_day_gap 已移除，所有 ts 算子用 shift(1)）
- 一字板过滤正常
- 交易成本已扣（佣金万 0.9 + 印花税万 5 + 滑点 0.1%）
- IC 与回测收益 Spearman 相关系数 -0.020（完全无关）
- 每日调仓，平均 100-200 笔/年

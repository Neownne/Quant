# ML 选股策略可解释性分析

> 日期：2026-05-26  
> 策略版本：DualPeriodModel v1 (日频量价ML + 月频基本面排雷)  
> 回测区间：2021-01 至 2026-05，800 只股票（沪深300+中证500）

---

## 一、策略架构概览

策略采用双周期结构，分离技术面预测与基本面风控：

```
每月首个交易日                    每日
┌─────────────────┐         ┌──────────────────────┐
│  基本面排雷      │         │  量价因子 ML 预测      │
│  fin_audit_score │  ───→   │  XGBoost+LightGBM     │
│  + 三项硬排除    │  白名单  │  概率平均集成          │
│  ~350/800 通过   │         │  Optuna 阈值调优       │
└─────────────────┘         └──────┬───────────────┘
                                   │ 横截面打分
                                   ▼
                          ┌──────────────────────┐
                          │  NDrop 增量调仓        │
                          │  Top-15 持有           │
                          │  每日淘汰 2 只 → 补入 2 只 │
                          └──────────────────────┘
```

**核心设计理念**：量价因子捕捉短期市场行为偏差（技术面 alpha），基本面排雷防止踩雷（财务质量 beta）。两者解耦使得策略失效时可以独立诊断。

## 二、因子体系的可解释性

### 2.1 因子分类与经济学含义

策略使用 **76 个因子**，分为 9 大类，每类有明确的经济学直觉：

| 类别 | 数量 | 经济学含义 | 代表性因子 |
|------|------|-----------|-----------|
| **Alpha101 核心** | 30 | 经典量化因子：动量、反转、波动率、流动性 | `mom_20`, `rev_5`, `vol_20`, `rsi_14` |
| **自定义A股** | 7 | A股特有结构特征：小盘溢价、筹码集中度、估值分位 | `log_mcap`, `sh_change`, `pb_pct` |
| **换手率** | 5 | 成交量的高阶统计特征：偏度、变异系数、相关性 | `turnover_skew`, `turnover_ret_corr` |
| **日内形态** | 4 | K线形态信息：上下影线、实体比例、日内反转 | `upper_shadow`, `lower_shadow`, `body_ratio` |
| **资金流向** | 6 | 量价关系推断资金方向：OBV、MFI、VWAP动量 | `money_flow`, `obv_roc`, `vwap_momentum` |
| **隔夜效应** | 4 | 开盘跳空与隔夜信息：隔夜收益、集合竞价跳跃 | `overnight_ret`, `open_auction_jump` |
| **波动率高阶** | 5 | 波动率结构特征：波动率波动、下行波动、尾部风险 | `vol_of_vol`, `tail_risk`, `down_vol_ratio` |
| **流动性高阶** | 4 | 微观结构：Amihud非流动性、价差代理、成交额 | `amihud_5`, `dollar_volume`, `bid_ask_proxy` |
| **基本面质量** | 11 | 财务健康度：盈利质量、成长性、杠杆、质押、商誉 | `fin_audit_score`, `fin_debt_ratio` |

### 2.2 IC 门禁：只保留有预测力的因子

每个因子计算逐日 **RankIC**（Spearman秩相关系数），衡量因子值与次日收益的单调关系：

```
通过条件：
  |mean IC| > 0.02   （预测力下限）
  |t-statistic| > 2.0 （统计显著性，等价于 ICIR × √N > 2.0）
```

在 800-stock 全市场回测中，65 个量价因子经 IC 门禁后保留 **20 个**（通过率 ~31%）。被淘汰的因子包括：
- beta_20 (|IC|=0.0013): A股个股与市场指数相关性弱
- kurtosis_20 (|IC|=0.0019): 峰度本身不具预测力
- pb_pct (|IC|=0.0086): 估值分位在日频无显著预测力
- gap_ma_dev (|IC|=0.0045): 缺口偏离均值不持续

这确保了进入模型的因子都具有**统计上显著且经济上有意义的预测力**。

### 2.3 正交筛选：去冗余、保留独立信号源

对通过 IC 门禁的因子，计算 Spearman 相关性矩阵，贪心选择相关系数 **< 0.7** 的因子：

```
20 个因子 → 正交筛选(threshold=0.7) → 8 个因子
```

最终入选的 8 个因子覆盖了不同类型的 alpha 信号源：
- **趋势/动量类**：ema_ratio_5_20, vwap_ratio
- **波动率类**：vol_20, atr_14
- **反转类**：rev_5, rsi_7
- **流动性类**：turnover_5, amount_ratio
- **资金流类**：force_index, vpt
- **形态类**：co_ratio, intra_day_rev
- **波动率结构**：vol_of_vol, tail_risk
- **A股结构**：log_mcap, sh_change

正交筛选的价值：8 个低相关因子组成的模型比 65 个高相关因子更稳健——减少多重共线性，降低过拟合风险，且每个因子提供独立的边际预测力。

### 2.4 特征重要性：模型级别的解释

XGBoost 和 LightGBM 均提供 `feature_importances_`（基于分裂次数或信息增益），可以在每个 walk-forward 窗口中排出最重要的因子。

这是一个**相对重要性排名**而非因果推断——它告诉我们"模型用了哪些因子做决策"，而非"哪些因子在现实中驱动了收益"。结合 IC 分析（统计相关性）和经济学直觉（因果关系），形成三层可解释性：

```
Layer 1: RankIC        → 因子与收益的统计相关性（是否有关？）
Layer 2: Feature Imp   → 模型依赖哪些因子做决策（用了什么？）
Layer 3: 经济学直觉     → 因子背后的行为金融逻辑（为什么有效？）
```

## 三、双周期设计的可解释性

### 3.1 月频基本面排雷

基本面包容 11 个因子，其中核心是 **fin_audit_score**（8项排雷综合评分）：

| 排雷项 | 触发条件 | 扣分 |
|--------|---------|------|
| 扣非净利润为负 | 主营业务亏损 | -1 |
| 利润为正但经营现金流为负 | 利润质量差 | -1 |
| 归母净利润为负 | 整体亏损 | -1 |
| ROE 为负 | 股东回报为负 | -1 |
| 净利润变异系数 > 1.0 | 盈利剧烈波动 | -1 |
| 净利率趋势下行 | 盈利能力恶化 | -1 |
| 资产负债率 > 70% | 杠杆过高 | -1 |
| 商誉/净资产 > 30% | 并购风险 | -1 |

**排雷逻辑**：每触发一项扣 1 分，0 分 = 完全健康，-8 = 全触发。

月频质量筛选：
- **阈值 -3.0**：允许触发最多 3 项（容忍度适中，~45% 股票通过）
- **硬排除**：商誉>27%/质押>72%/负债>105%（极端风险零容忍）

在 800 只候选股票中，每月约 **330-392 只** 通过质量筛选，排除率约 **51-59%**。

### 3.2 日频 ML 预测

XGBoost + LightGBM 概率平均集成，Optuna 自动搜索最优分类阈值（最大化验证集 F1）。

Walk-forward 滚动训练（3 年训练 + 1 年验证）确保模型在任何时点只使用历史数据，无前视偏差。

### 3.3 为什么双周期有效？

- **量价因子日频变化快**（动量/反转/波动率），需要每日更新预测
- **基本面因子变化慢**（财报季频），月度更新即可，且日频更新无增量信息
- **解耦设计**：如果策略失效，可以独立判断是 ML 选股问题还是基本面筛选问题

## 四、交易执行层的可解释性

### 4.1 NDrop 增量调仓

传统 Top-K 策略每日全换仓，换手率极高且不稳定。NDrop 算法：

```
1. 每日 ML 预测横截面打分
2. 排序取 Top-K 作为目标池
3. 对比当前持仓 vs 目标池，只替换差异部分
4. 每期最多替换 N 只（K=15, N=2）
```

**直觉**：好的 alpha 信号不会一天就消失。适度保留现有持仓减少换手，降低交易成本，同时保持对信号变化的响应。

网格搜索结果验证了这一直觉：NDrop(ndrop_n=2) 在所有 top_n 级别都优于全换仓。

### 4.2 指数崩盘过滤器

```
中证1000 近 15 日累计跌幅 ≤ -12% → 全部清仓
```

这保护策略在市场系统性暴跌时不承担多头风险。15 日窗口足够短以快速响应，-12% 阈值在A股历史上约每年触发 1-2 次。

## 五、Walk-Forward 稳健性验证

### 5.1 滚动窗口结构

```
窗口 1: 训练[2021-2023] → 验证[2024] → 模拟[2024]
窗口 2: 训练[2022-2024] → 验证[2025] → 模拟[2025]
窗口 3: 训练[2023-2025] → 验证[2026] → 模拟[2026]
...
```

每个窗口独立训练、独立验证，最终表现是 **多窗口平均**，而非单次拟合。

### 5.2 过拟合检测

| 指标 | 样本内 (2019-2022) | 样本外 (2024-2026) |
|------|:---:|:---:|
| Sharpe | 1.09 | **2.27** |
| 年化收益 | 20.2% | 42.0% |
| 最大回撤 | -28.5% | -31.5% |
| 胜率 | 55.0% | 57.9% |

**衰减比 = 208.1%** — 样本外表现反而优于样本内。这说明：
1. IC 门禁 + 正交筛选有效去除了噪音因子
2. Walk-forward 训练避免了过拟合历史模式
3. 基本面排雷在样本外期同样有效（2024-2026 年A股基本面分化加剧）

## 六、参数敏感度分析

50 只股票网格搜索（12 组参数组合）：

| top_n | NDrop | ndrop_n | Sharpe | 解读 |
|-------|-------|---------|--------|------|
| 15 | Y | 2 | **0.90** | 最优：适度集中+温和调仓 |
| 20 | N | - | 0.78 | 分散但高换手 |
| 15 | Y | 1 | 0.73 | 调仓不足 |
| 15 | N | - | 0.72 | 集中+高换手 |
| 10 | Y | 2 | 0.59 | 过于集中 |
| 10 | Y | 3 | 0.47 | 集中+过度调仓 |

**规律**：
- top_n=15 是甜点区：10 太集中（少数股票出错影响大），20 太分散（alpha 稀释）
- NDrop 在各级别都优于全换仓
- ndrop_n=2 最优：1 太少（响应慢），3 太多（接近全换仓）

## 七、策略局限性 & 可解释性边界

### 7.1 当前可解释的

- **因子层面**：每个因子有明确的经济学含义和 IC 统计量
- **筛选层面**：IC 门禁和正交筛选的去留标准透明可审计
- **模型层面**：树模型的特征重要性排名提供相对重要性
- **组合层面**：NDrop 逻辑、止损阈值、行业上限均有显式规则

### 7.2 当前不够可解释的

- **SHAP 值缺失**：无法量化单个因子对单次预测的边际贡献
- **交互效应不透明**：XGBoost/LightGBM 自动学习因子交互，但未显式建模
- **特征重要性偏差**：树模型的默认 importance 偏向高基数特征
- **概率输出未校准**：模型输出的是排序分数而非概率
- **行业覆盖率不完整**：沪市/北交所 ~58% 股票无行业标签，行业上限仅部分生效

### 7.3 后续改进方向

1. **SHAP 分析**：计算每个预测的 SHAP 值，解释"为什么选这只股票"
2. **部分依赖图 (PDP)**：可视化因子与预测概率的非线性关系
3. **因子归因**：将组合收益分解为因子收益 + 特质收益
4. **置换重要性**：比树模型默认 importance 更无偏
5. **行业数据补充**：使用申万行业分类 API 或 Wind 数据补全沪市行业标签

## 八、策略失效场景

| 场景 | 触发条件 | 策略行为 |
|------|---------|---------|
| 市场崩盘 | 中证1000 15日跌超12% | 全部清仓，等待信号恢复 |
| 个股闪崩 | 单日跌超8% 或 ATR止损 | 清仓该股 |
| 组合回撤 | 总资产较高点回撤超20% | 减仓50% |
| 最大回撤 | 总资产较高点回撤超25% | 全部清仓 |
| 风格切换 | 动量因子失效 | Walk-forward 自动降低失效因子权重 |
| 财务暴雷 | 触发 audit_score 排雷 | 月频被移出候选池 |

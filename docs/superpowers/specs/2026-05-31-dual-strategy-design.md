## 小市值策略规则

### 候选池
- 最小市值300只（排除688/300/4/8、ST、上市<375天）
- 每月初刷新

### 入场（全部满足才买入）
1. 今日开盘 < 昨日最低价
2. 开盘跌幅 > 0.75%
3. 近4日振幅 < 10%
4. 昨收 > 5日均线
5. 昨日非涨停
6. 行业分散：每行业最多1只
7. 持仓未满（默认10只等权）

### 出场
1. 硬止损 -3%
2. 收盘价 < 开盘价 → 卖出
3. 线性移动止盈（涨1%→容忍0.5%回撤，涨10%→容忍2%）
4. 跌停不卖（延至次日）

### 季节空仓
- 1月/4月/12月20-31日/3月20-31日

### 执行时点
- 入场：T日收盘信号 → T+1日开盘价成交
- 出场：T日收盘信号 → T+1日开盘价成交
